# Roadmap

## Phase 0 — Prototype

- Demonstrate persistent behavioral divergence
- Validate that separate interaction histories produce different outputs
- Establish basic state-driven behavior

Status: In progress / demonstrated in prototype

## Phase 1 — Modularization

- Separate core system components
- Build repeatable test harnesses
- Document public architecture at a safe abstraction level

## Phase 2 — Production Proof of Concept

- Integrate with production LLM APIs
- Improve state management
- Create live demo environment
- Add behavior comparison tooling

## Phase 3 — Partner / Investor Demo

- Build a controlled demo showing two instances with different histories
- Track measurable divergence
- Prepare for technical review with qualified collaborators
